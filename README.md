# neteaseMusic(又是一个网易客户端)

[http://music.otar.im/](http://music.otar.im/)

**功能**

1. 支持歌单，单曲，专辑，艺人，用户搜索
2. 支持歌曲清单
3. 支持快捷键，桌面通知
4. 更多功能持续开发中

**使用**

`npm install` + `bower install` 安装所需的第三方依赖, `npm start` 启动 app,默认监听 `9527` 端口
